## ğŸš€ Getting Started

### Prerequisites

-   **Node.js** (v18+)
-   **pnpm** (Recommended) or npm/yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Ajvadn/portfolio-ajvad.git
    cd portfolio-ajvad
    ```

2.  **Install dependencies**
    ```bash
    pnpm install
    ```

3.  **Start the dev server**
    ```bash
    pnpm run dev
    ```
    Access the site at `http://localhost:4321`
